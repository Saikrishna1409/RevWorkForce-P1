The RevWorkforce is a Java-based console application designed to streamline employee, manager, and admin operations in an organization. It provides functionalities for employee management, leave management, performance tracking, goal setting, and reporting.The system follows a modular architecture using DAO-Service-Controller patterns and incorporates role-based access, ensuring that employees, managers, and admins have appropriate permissions and functionality.It uses MySQL as the backend database and Log4j2 for logging. The system also includes input validation, password hashing, and audit logging to ensure security and maintainability.


Technologies Used

Language: Java 17
Database: MySQL 8.x
Logging: Log4j2
Build Tool: Maven (if applicable)
Password Security: BCrypt
Architecture Pattern: MVC + DAO-Service
Version Control: Git & GitHub
